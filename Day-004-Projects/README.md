# Day 4 - Rock Paper Scissors Game

## Project Overview

Today I built the classic **Rock, Paper, Scissors** game using Python. The player competes against the computer, which randomly selects its move. The game then determines the winner based on the standard rules.

## Concepts Practiced

* Variables
* Lists
* User Input
* Conditional Statements (`if`, `elif`, `else`)
* Random Module (`random.choice()`)
* Input Validation
* Program Logic

## How the Game Works

1. The player chooses:

   * 0 = Rock
   * 1 = Paper
   * 2 = Scissors
2. The computer randomly selects one of the three options.
3. The game compares both choices and displays whether the player wins, loses, or ties.

## What I Learned

* Using Python's `random` module to generate unpredictable outcomes.
* Storing related data in lists.
* Validating user input to prevent errors.
* Building complete game logic using conditional statements.
* Creating interactive command-line games.

## Future Improvements

# Day 7 - Hangman Game

## Project Overview

Today I built the classic **Hangman** game using Python.

The player tries to guess a hidden word one letter at a time. Each incorrect guess reduces the number of lives. The game ends when the player successfully guesses the word or runs out of lives.

## Concepts Practiced

* Variables
* Lists
* Loops (`while` and `for`)
* Conditional Statements (`if`, `elif`, `else`)
* Python Modules (`import`)
* Random Module
* String Methods
* Game Logic
* Tracking User Input

## Features

* Random word selection
* ASCII art for the game logo and hangman stages
* Tracks previously guessed letters
* Prevents duplicate guesses
* Updates the hidden word after every correct guess
* Displays remaining lives
* Win and lose conditions

## What I Learned

This project taught me how to combine multiple Python concepts into one complete application.

I practiced:

* Organizing code across multiple files using modules.
* Managing the state of a game.
* Validating user input.
* Writing cleaner program logic.
* Building an interactive command-line application.

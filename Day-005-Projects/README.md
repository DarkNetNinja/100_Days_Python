# Day 5 - Password Generator

## Project Overview

Today I built a **Password Generator** using Python. The program allows the user to choose how many letters, numbers, and special symbols should be included in the password, then generates a randomized password.

## Concepts Practiced

* Variables
* Lists
* Loops (`for`)
* User Input
* String Manipulation
* Random Module (`random.choice()` and `random.randint()`)
* Password Generation Logic

## How the Program Works

1. The user enters:

   * Number of letters
   * Number of numbers
   * Number of special symbols
2. The program randomly selects characters from predefined lists.
3. All characters are combined.
4. The password is randomized before being displayed.

## What I Learned

* Building a real-world utility with Python.
* Generating random values using the `random` module.
* Working with strings and lists together.
* Using loops to automate repetitive tasks.
* Designing program logic based on user input.

## Possible Improvements

* Use `random.shuffle()` to randomize characters more efficiently.
* Prevent duplicate character selection if required.
* Allow users to choose password strength (Easy, Medium, Strong).
* Add an option to generate multiple passwords at once.
* Create a graphical version using Tkinter.

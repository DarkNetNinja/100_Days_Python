# Day 3 - Treasure Island Game

## Project Overview

Today I built **Treasure Island**, a text-based adventure game in Python where the player makes choices to find the hidden treasure. Different decisions lead to different outcomes, including winning the game or reaching a game over.

## Concepts Practiced

* Variables
* User Input
* Conditional Statements (`if`, `elif`, `else`)
* Nested Conditional Statements
* String Methods (`.lower()`)
* Program Flow and Decision Making

## How the Game Works

1. Choose a direction at the crossroads.
2. Decide whether to wait for a boat or swim across the lake.
3. Choose one of three doors.
4. Your choices determine whether you find the treasure or lose the game.

## What I Learned

* How to build interactive command-line applications.
* Writing nested `if-else` statements.
* Controlling program flow based on user input.
* Making programs more user-friendly by handling uppercase and lowercase input with `.lower()`.

## Future Improvements

* Add more story paths.
* Create an inventory system.
* Keep track of player health and score.
* Convert the game into a graphical version using Python libraries.
